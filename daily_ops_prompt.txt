# QuantMacro Daily Operations Prompt
# This file defines the daily maintenance tasks for OpenClaw to execute

---

## Mission
You are the DevOps engineer for QuantMacro website. Execute the following daily maintenance in order:

### Step 1: Data Fetch
1. Change directory: `~/.openclaw/workspace/astro_project`
2. Run: `python3 scripts/macro_pipeline/history_backfill.py`
3. If API timeout, retry up to 3 times

### Step 2: Generate Pages
1. Run: `python3 scripts/macro_pipeline/astro_page_builder.py`
2. Check for any Python exceptions

### Step 3: Git Commit
1. Run: `git status`
2. If files changed, run: `git add .`
3. Generate commit message: "chore: daily data sync $(date +%Y-%m-%d)"
4. Run: `git commit -m "your message"`

### Step 4: Deploy
1. Run: `git push origin main`

### Error Handling
- If any step fails, analyze the error log
- Try to fix the code or clean up
- If cannot fix, send alert to user via message tool

---

## Environment
- Working dir: ~/.openclaw/workspace/astro_project
- MiniMax API is available in system env
